# AR Character Placement App

A simple Unity-based AR project developed for a course assignment.

## Features
- Scans a real-world area
- Places a selected 3D character into the environment
- Allows basic interaction on mobile

## Technologies
- Unity
- C#
- AR

## Description
This project was developed for a Visual and Environmental Design course. It allows users to scan a surface, place a selected character in the environment, and view it through a mobile device.
